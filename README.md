# 随机密码生成器

一个简单优雅的在线密码生成工具，可以生成6位数字密码、12位密码（简单模式：大小写字母+数字；复杂模式：大小写字母+数字+特殊字符）和18位密码。

## 本地部署

1. 克隆仓库
```bash
git clone https://github.com/lovexw/random-password.git
```

2. 打开index.html文件即可使用

## 部署到Cloudflare Pages

1. 登录Cloudflare账户
2. 进入Pages页面，点击"创建项目"
3. 连接您的Git仓库
4. 选择项目根目录
5. 构建设置：
   - 构建命令：留空
   - 构建输出目录：留空
6. 点击"保存并部署"
7. 等待部署完成即可访问

## 项目特点

- 完全客户端运行，无服务器交互
- 支持多种密码强度
- 一键复制功能
- 响应式设计
